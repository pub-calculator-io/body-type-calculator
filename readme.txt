[https://www.calculator.io/body-type-calculator/](https://www.calculator.io/body-type-calculator/)

== Usage ==

Add the Body Type Calculator shortcode to your page, post or sidebar:

`[ci_body_type_calculator]`

Add the following code to your template where you would like the Body Type Calculator to appear:

`<?php display_ci_body_type_calculator(); ?>`

1. Upload the Body Type Calculator /ci_body_type_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Body Type Calculator plugin through the "Plugins" menu in WordPress.
